# 💻 Mini Shell (C)

A simple Unix-like shell built in C that executes user commands, supports piping, and demonstrates process handling.

## 🚀 Features

* Execute Linux commands
* Support for multiple commands with piping
* Process creation using fork()
* Command parsing and execution
* Basic shell prompt

## 🛠️ Tech Used

C, System Calls, fork(), exec(), pipe(), File Descriptors

## ⚙️ Run

```bash
make
./a.out
```
